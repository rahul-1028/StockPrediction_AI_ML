# 📈 Stock Prediction AI
This project predicts stock prices using **Machine Learning (Linear Regression) & Deep Learning (LSTM)**.

## 🚀 Features:
- 📊 Fetches stock data from **Yahoo Finance API**
- 🧠 Predicts future stock prices
- 🎨 Visualizes historical stock trends
